# Movie Review System

## Description
A Spring Boot-based backend application for managing movie reviews and tickets.

## Features
- Add movie reviews
- View all reviews
- Book tickets
- Backend REST API
- Database integration

## Tech Stack
- Java 17
- Spring Boot
- Maven


## How to Run
1. Clone the repository
2. Open in Eclipse or IntelliJ
3. Run the main Spring Boot application
